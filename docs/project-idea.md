# Идея проекта: Foundation of Code

## Основная концепция

Создать образовательный сайт, который объединяет фундаментальные концепции программирования и объясняет, как эти концепции реализуются в разных языках.

## Основные особенности

- **Обучение от абстракции к реализации.**
  Пример: память → как она работает в C → что переносится в Python → что меняется в JS.
- **Связи между языками.**
- **Наглядные диаграммы и визуальные карты.**
- **Роудмапы обучения для разных целей.**
- **Контекстный форум / комментарии.**

## Проблема, которую решает проект

Большинство ресурсов дают информацию фрагментированно: отдельно языки, отдельно теория. Новички путаются и не понимают, что общее между разными технологиями.

## Чем уникален проект

- общая платформа фундаментальных концепций  
- сравнение языков между собой  
- сочетание статей, визуализаций и дискуссий  
- единая база знаний + форум  

## Целевая аудитория

- начинающие программисты  
- студенты  
- люди, переходящие с одного языка на другой  
- самоучки
