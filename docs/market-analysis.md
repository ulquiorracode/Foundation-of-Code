# Анализ рынка и конкурентов

## Таблица конкурентов

| Сайт | Описание | Сильные стороны | Ограничения |
|------|----------|------------------|--------------|
| W3Schools | Учебник по языкам веб-разработки | простота, живые примеры, песочница | нет теории, нет связи между языками |
| MDN Web Docs | Документация Mozilla | глубина, качество, точность, примеры | сложно новичкам, акцент на веб |
| Codecademy | Интерактивные курсы | практика, мини-проекты, структурированность | платный контент, мало фундаментальной теории |
| Хабр | Статьи от специалистов | разнообразие, живые примеры | нет структуры, уровень очень разный |
| LearnCPP / LearnPython и др. | Учебники по отдельным языкам | системность внутри одного языка | нет объединяющей концепции |

## Выводы

— Ни один ресурс не предлагает **единую систему фундаментальных концепций**, показывая их реализацию в разных языках.  
— Отсутствует **“сквозное обучение”: от темы → к практике → к обсуждению**.  
— Foundation of Code закрывает эти пробелы.
