# Структура контента Foundation of Code

## Разделы сайта

### 1. Базовые концепции

- память
- переменные и область видимости
- типы данных
- условия
- циклы
- структуры данных
- функции
- работа с файлами
- обработка ошибок

Каждая статья включает:

- теоретическую часть
- отсылки в конкретных роудмапах по языкам программирвоания
- форум/комментарии, ссылающиеся на статью

### 2. Языки программирования

- C#
- Python  
- Java
- позже: Go, Rust, JavaScript (язык инвалидов) и другие

### 3. Роудмапы

- «От нуля до джуна»
- «Понимание низкоуровневых концепций»
- «Переход с Python на C»

### 4. Форум / обсуждения

- привязанные к статьям ветки  
- общие топики
