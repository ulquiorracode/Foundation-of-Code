# Foundation of Code

Централизованная образовательная платформа, где собраны фундаментальные основы программирования, связи между языками, визуальные материалы, обучающие траектории и база знаний с возможностью обсуждения.

Этот репозиторий содержит:

- документацию проекта,
- анализ рынка,
- структуру будущего веб-сайта,
- материалы для презентации,
- распределение ролей,
- технические решения и стек.

## Цель проекта

Создать удобный, структурированный и понятный образовательный ресурс, который связывает фундаментальные концепции программирования с конкретными языками и предоставляет пользователям визуальные и интерактивные инструменты для обучения.

## О проекте

Foundation of Code — это не просто набор статей. Это:

- системное объяснение базовых концепций,
- сравнение разных языков через призму фундаментальных идей,
- визуальные карты, схемы и интерактивные примеры,
- тематические роудмапы,
- комментарии и обсуждения под каждой статьей.

Подробнее — см. [project-idea](docs\project-idea.md)

## MVP включает

- главную страницу + меню
- раздел «Базовые концепции»
- 5–10 фундаментальных тем → 3 языка (C, Python, JS)
- базовый форум/комментарии
- таблицу сравнений языков
